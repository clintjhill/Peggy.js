Example: Additive - Easy

The following is an example of creating a grammar that will parse out a very simple addition calculation. It is simple in that it does not respect whitespace and does not account for floating-point numbers.

Grammar

The first step is to create the grammar. The next step is to provide the grammar with a root rule that will act as the entry-point for the parsing.

var sample = new Peggy('sample');

sample.root('additive', sample.sequence(/\d+/, '+', /\d+/), function(value){
	return new Number(value[0]) + new Number(value[2]);
});

The rule "additive" above simply provides a sequence of terminal rules which are un-named. The first and last are regular expressions that will look for a numeral and the second is going to look for a + symbol. The function is used as an extension to the rule and will receive the value of the parse as an argument.

  • 4567+9876
