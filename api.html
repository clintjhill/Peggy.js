Peggy API

Peggy

This object is the core constructor of grammars. Instatiating this provides all of the rule building functionality and includes the rule engine.

Example
var grammar = new Peggy('sample');
peggy.rule(name, declaration, [extension])
The main rule building function. name and declaration are required while extension is optional.
peggy.root(name, declaration, [extension])
Identical to peggy.rule with the added functionality of marking this rule as the entry-point for the entire grammar.
peggy.sequence(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules to be used as a declaration for a rule.
