Peggy API

Peggy

This object is the core constructor of grammars. Instatiating this provides all of the rule building functionality and includes the rule engine.

Example
var grammar = new Peggy('sample');
peggy.rule(name, declaration, [extension])
The main rule building function. Creates a terminal rule for the grammar.
  • name - string to identify this rule within the grammar and within value objects
  • declaration - the body of the rule. Can be a Regular Expression, a string terminal or an :alias to another rule
  • extension - optional function to be executed against the result of the rule.
grammar.rule('number', /\d+/, function(value){
      return new Number(value);
}
peggy.root(name, declaration, [extension])
Identical to peggy.rule with the added functionality of marking this rule as the entry-point for the entire grammar.
peggy.any(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that is succesful on "any" match on the string.
peggy.all(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that require a match on "all" parts of the rule.
peggy.choice(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that require a match on only one part of the rule.
peggy.oneOrMore(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that will repeat a match on "one or more" parts of the rule.
peggy.zeroOrMore(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that will repeat a match on "zero or more" parts of the rule.
peggy.zeroOrOne(rule, [rule2], [rule3], [...])
Creates a non-terminal collection of rules, that will repeat a match on "zero or one" part of the rule.
peggy.parse(string)
Parses the provided string against the root rule of the grammar and returns the value of that rule.
:alias
A string prefixed by a colon that is a reference to a previously named rule within the grammar.
extension
A function provided to a rule that is used to further the evaluation of the rule. It receives a value parameter that is the object representation of the rule. 
function(value) {
      // if only one rule declaration
      var r = value;
      // un-named declarations (ordinal position)
      var u = value[0];
      //:number provided in the rule declaration
      var n = value.number; 
      //:number used in multiple places in declaration
      var narr = value.number[0];
}
